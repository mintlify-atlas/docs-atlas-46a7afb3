## Built With Modern Technology

Beils Dashboard leverages cutting-edge technologies for optimal performance:

- **Nuxt 4** - Full-stack Vue framework with server-side rendering
- **Tailwind 4** - Modern utility-first CSS framework
- **Prisma** - Type-safe database ORM with MariaDB
- **Vue 3** - Progressive JavaScript framework
- **TypeScript** - Type-safe development experience

<Note>
  Beils Dashboard follows the "belleza honesta" (honest beauty) philosophy, providing transparency and ethical practices in beauty center management.
</Note>
