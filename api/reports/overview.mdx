## Overview

The reports overview endpoint provides aggregated analytics data for your beauty center operations. This endpoint returns key performance indicators (KPIs), revenue trends over the last 6 months, payment method distribution, and top-selling items.

<Note>
This endpoint aggregates data from completed sales only (carts with status `completed`).
</Note>

## Authentication

This endpoint requires authentication. Include a valid Bearer token in the Authorization header.

```bash
Authorization: Bearer YOUR_TOKEN
```

## Request

<ParamField query="none" type="none">
  This endpoint does not accept any query parameters.
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://your-domain.com/api/reports \
  --header 'Authorization: Bearer YOUR_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://your-domain.com/api/reports', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://your-domain.com/api/reports"
headers = {"Authorization": "Bearer YOUR_TOKEN"}

response = requests.get(url, headers=headers)
data = response.json()
print(data)
```

</CodeGroup>

## Response

The endpoint returns a JSON object containing four main sections: KPIs, revenue trend, payment methods distribution, and top-selling items.

<ResponseField name="kpis" type="object">
  Key Performance Indicators summarizing overall business metrics
  
  <Expandable title="KPIs Object">
    <ResponseField name="totalRevenue" type="number">
      Total revenue from all completed sales
    </ResponseField>
    
    <ResponseField name="totalClients" type="number">
      Total number of clients (users with role 'USER')
    </ResponseField>
    
    <ResponseField name="totalProducts" type="number">
      Total number of products in the catalog
    </ResponseField>
    
    <ResponseField name="totalSales" type="number">
      Total number of completed sales transactions
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="revenueTrend" type="array">
  Monthly revenue data for the last 6 months
  
  <Expandable title="Revenue Trend Item">
    <ResponseField name="month" type="string">
      Month label in uppercase Spanish format (e.g., "ENE", "FEB", "MAR")
    </ResponseField>
    
    <ResponseField name="revenue" type="number">
      Total revenue for the specified month
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="paymentMethods" type="array">
  Distribution of payment methods used across all completed sales
  
  <Expandable title="Payment Method Item">
    <ResponseField name="name" type="string">
      Payment method name. Possible values:
      - `Efectivo` (Cash)
      - `Tarjeta` (Card)
      - `Mixto` (Mixed)
      - `Transferencia` (Transfer)
    </ResponseField>
    
    <ResponseField name="value" type="number">
      Number of transactions using this payment method
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="topItems" type="array">
  Top 5 best-selling items by quantity sold
  
  <Expandable title="Top Item">
    <ResponseField name="name" type="string">
      Name of the product, service, pack, or item
    </ResponseField>
    
    <ResponseField name="quantity" type="number">
      Total quantity sold across all completed sales
    </ResponseField>
  </Expandable>
</ResponseField>
